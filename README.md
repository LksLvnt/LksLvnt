## Featured Projects

### [StudyMate](https://github.com/LksLvnt/StudyMate)
AI-assisted study tool with a retrieval-augmented generation pipeline over user notes and spaced-repetition scheduling. React 19 frontend, FastAPI backend, PostgreSQL with pgvector, Claude API. Dockerized with automated CI/CD via GitHub Actions.
Live: [studymate.lokoslevente.com](https://studymate.lokoslevente.com)

### [UniTools](https://github.com/LksLvnt/UniTools)
Student productivity app: timetable manager, Hungarian grade calculator, and Pomodoro timer. Angular with Signals, Supabase for real-time data persistence.

### [lokoslevente.com](https://github.com/LksLvnt/Actual-Website)
Personal portfolio site. Astro, TypeScript, Tailwind CSS, EN/HU internationalization, continuous deployment via GitHub Actions.
Live: [lokoslevente.com](https://lokoslevente.com)

### [coffee-portfolio](https://github.com/LksLvnt/coffee-portfolio) — in progress
Interactive Three.js 3D café scene built with Vite, TypeScript, and GSAP, where each stage of making espresso represents a section of the portfolio.
